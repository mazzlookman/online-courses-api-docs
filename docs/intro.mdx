# Overview

### Online Courses API Documentation

Dokumentasi API untuk proyek Online Course yang dibangun menggunakan Laravel bertujuan untuk memberikan panduan bagi yang ingin mengintegrasikan atau memanfaatkan layanan ini. API ini menyediakan berbagai endpoint untuk mengelola kursus online, termasuk fitur-fitur seperti pendaftaran pengguna (baik sebagai mentor, peserta, maupun admin), autentikasi dan otorisasi menggunakan token, pengelolaan kursus, pelajaran, konten multimedia, serta transaksi pembelian.

Setiap endpoint disertai dengan semantic HTTP method, parameter yang diperlukan, serta contoh request dan response, sehingga mempermudah developer dalam memahami dan mengimplementasikan API ini ke dalam aplikasi mereka.

# Register User

Endpoint ini digunakan untuk registrasi user baru.

### Path

import MethodBox from "../../src/components/MethodBox/MethodBox";

<MethodBox method="POST" endpoint="/users" methodColor="#48CB90" />


### Request Parameters

| Parameter | Type | Required | Description                            |
|-----------| --- |----------|----------------------------------------|
| name      | string | required | Nama user                              |
| email     | string | required | Email aktif user                       |
| password  | string | required | Password user                          |
| avatar | string | optional | Foto profile user                      |
| role | string | required | Role user (user/admin), default "user" |

### Example Request

```json
{
  "name": "Bella Rizky Kharisma",
  "email": "bella@test.com",
  "password": "123",
  "avatar": "bella.jpg",
  "role": "admin"
}
```

### Response Parameters

| Name | Type | Description                     |
|------|------|---------------------------------|
| id | int  | Id user yang berhasil registrasi |
| name | string | Nama user                       |
| email | string | Email user                      |
| avatar | string | Foto profile user               |
| role | string | Role user (admin/user)          |
